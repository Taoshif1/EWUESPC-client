# 🎮 EWUESPC Client

Official frontend for **EWUESPC**  
(**East West University E-Sports Club**)

A modern, minimal, and scalable esports club website built with **React 19**, **Vite**, **React Router v7**, **Tailwind CSS v4**, and **DaisyUI v5**.

This project is being developed with a **professional-grade MERN architecture mindset**, focusing on **clean UI**, **future scalability**, **theme support**, and **modern frontend engineering practices**.

---

## 🚀 Project Vision

EWUESPC is designed to be more than just a student club website.

The goal is to build a digital platform that can eventually support:

- Club branding and identity
- Event and tournament showcase
- Team / executive panel display
- Member onboarding / join requests
- Club gallery and achievements
- Sponsor / partner visibility
- Admin and dashboard expansion in the future

---

## 🛠️ Tech Stack

### Frontend Core
- **React 19**
- **Vite**
- **React Router v7**
- **Tailwind CSS v4**
- **DaisyUI v5**

### Additional Libraries
- **Framer Motion** → animations
- **React Helmet Async** → dynamic page titles / SEO
- **TanStack Query** → future server state management
- **Axios** → API communication
- **React Hook Form** → form handling
- **SweetAlert2** → alert modals
- **Lucide React** → icons
- **React Icons** → icon library
- **clsx** → conditional class handling

## ⚙️ Installation & Setup

### 1) Clone the repository

```bash
git clone <your-repo-url>
```

### 2) Go into the client folder

```bash
cd client
```

### 3) Install dependencies

```bash
npm install
```

### 4) Start the development server

```bash
npm run dev
```
